# Prompt para continuar con otra IA

Use este prompt si va a pedirle a otra IA que continue el trabajo.

```text
Actua como ingeniero senior y auditor de producto clinico IAAS. Trabaja en la carpeta actual del proyecto `APLICACION MAMA`.

Antes de responder o modificar, lee en este orden:
1. AGENTS.md
2. TRASPASO_PROGRESIVO_IAS.md
3. ESTADO_INTERVENCION_AUDITORIA.md
4. REGISTRO_PROGRESO_IAS.md
5. README_RUN.md
6. PROTOCOLO_VALIDACION_CLINICA.md
7. POLITICA_PRIVACIDAD_OPERATIVA.md
8. VERSIONAMIENTO_CRITERIOS_IAAS.md

Reglas:
- No empieces de cero.
- No omitas limitaciones ni pendientes.
- No digas que el sistema esta listo para piloto sin validacion.
- No uses IA externa ni datos reales salvo instruccion explicita.
- Mantén `Seguro local` como modo predeterminado.
- Toda confirmacion IAAS debe tener evidencia trazable.
- Todo resultado requiere revision humana.
- Actualiza REGISTRO_PROGRESO_IAS.md despues de cambios relevantes.

Objetivo:
Continuar mejorando progresivamente las areas auditadas: seguridad clinica, privacidad, trazabilidad, criterios, validacion, reportes, UX, empaquetado y preparacion para piloto.

Primero reporta brevemente:
- Que archivos leiste.
- Que area auditada vas a intervenir.
- Que cambio haras.
- Que validacion queda posible.
```
